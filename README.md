# Shopify Akeneo App

This is a Shopify app to synchronize products from Akeneo PIM to Shopify.

## Setup

To use the app, you need to configure the Akeneo credentials in the app's settings.

1.  Go to the app's home page.
2.  Click on the "Settings" menu item.
3.  Fill in the following fields with your Akeneo API credentials:
    *   **Akeneo URL**: The URL of your Akeneo instance (e.g., `https://my-akeneo-instance.com`).
    *   **Client ID**: Your Akeneo API client ID.
    *   **Client Secret**: Your Akeneo API client secret.
    *   **Username**: Your Akeneo username.
    *   **Password**: Your Akeneo password.
4.  Click "Save".

## Entities

The app allows you to view and manage the following Akeneo entities:

*   **Products**: View a list of your Akeneo products.
*   **Categories**: View your Akeneo category trees.
*   **Attributes**: View your Akeneo attributes.
*   **Product Families**: View your Akeneo product families.

You can navigate to each of these pages from the app's main navigation.
